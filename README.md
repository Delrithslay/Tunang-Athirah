# Jemputan Tunang Athirah — static invitation

Files added:

- `index.html` — main page
- `styles.css` — styling
- `script.js` — gallery + audio playback (WebAudio fallback)
- `assets/images/` — sample SVG images (`invitation.svg`, `floral.svg`)
- `assets/music/README.txt` — place `music.mp3` here to use a real audio file

How to use:

1. Open `index.html` in a browser.
2. Click thumbnails to view larger images.
3. Click "Play Music" to play the local `assets/music/music.mp3` if present — otherwise a brief synthesized tune will play.
